tornado-sessions
================

Multi-Backend Session Support for Tornado

Backends
--------

* MongoDB (PyMongo)

  * PyMongo
  * motor
  
* Redis

  * redis
  * tornado-redis
  
* MySQL

  * MySQLdb
  * adisp
  
* PostgreSQL

  * psychopg2
  * adisp 

* Cookies

  * standard
  * Secure

* Encrypted Cookies

  * tornado-encookie
